grep -l *PATTERN* *FILENAME*：FILENAMEからPATTERNを検索する, 検索結果にファイル名のみ表示する

!grep -l linux README.md

grep: README.md: No such file or directory

grep -c *PATTERN* *FILENAME*：FILENAMEからPATTERNを検索する, 検索結果に一致した箇所から前後に指定した行数表示する

!grep -c linux README.md

grep: README.md: No such file or directory

grep -c *PATTERN* *FILENAME*：FILENAMEからPATTERNを検索する, 検索結果に行番号を表示する

!grep -n linux README.md

grep: README.md: No such file or directory

grep -e *PATTERN* *FILENAME*：FILENAMEからPATTERNを検索する, 一致処理に指定した正規表現を使う

!grep -e linux README.md

grep: README.md: No such file or directory

grep -v *PATTERN* *FILENAME*：FILENAMEからPATTERNを検索する, 一致しないものを検索する

!grep -v linux README.md

grep: README.md: No such file or directory

grep -r *PATTERN* *FILENAME*：FILENAMEからPATTERNを検索する, <span style='background:blue'>ディレクトリ内も検索対象とする</span>

!grep -r linux src/*

grep: src/*: No such file or directory
